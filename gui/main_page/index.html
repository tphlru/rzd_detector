Как это работает?

Благодаря написанию frontend части кода на языке разметки HTML, наш продукт доступен на общедоступном сайте, где обработка данных происходит на сервере, а устройство пользователя лишь отправляет данные и отображает результаты.

Безопасность

LaboRad гарантирует не использовать биометрические, физические и любые другие данные пользователей для их распространения или передачи третьим лицам. Мы оставляем за собой право использовать данные пользователей только для улучшения работы этого приложения и обучения своих нейросетей.

Интерфейс

Специально разработанный для максимально легкого и удобного использования. Разработаны несколько версий для разных устройств.
